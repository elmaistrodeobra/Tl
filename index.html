Tablas y Listas en HTML

HTML proporciona elementos para organizar la información en tablas y listas. A continuación, se muestran ejemplos de cómo crear cada uno de estos elementos:

Tablas en HTML

Las tablas se utilizan para organizar datos en filas y columnas. Los elementos principales de una tabla son:

  • <table>: Define el contenedor de la tabla.
  • <tr>: Define una fila de la tabla.
  • <th>: Define una celda de encabezado (negrita y centrada por defecto).
  • <td>: Define una celda de datos.

Listas en HTML

HTML ofrece tres tipos principales de listas:

  • <ul>: Lista desordenada, donde cada ítem está marcado con un punto.
  • <ol>: Lista ordenada, donde cada ítem está numerado.
  • <dl>: Lista de definición, usada para pares de términos y descripciones.

Lista Desordenada (<ul>)

Una lista desordenada se utiliza cuando el orden de los elementos no es importante:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Lista Ordenada (<ol>)

Una lista ordenada se utiliza cuando el orden de los elementos es importante:

  1. Primer paso
  2. Segundo paso
  3. Tercer paso

Lista de Definición (<dl>)

Una lista de definición se utiliza para pares de términos y descripciones:

HTML
Lenguaje de marcado utilizado para crear páginas web.
CSS
Lenguaje de estilos utilizado para diseñar páginas web.
JavaScript
Lenguaje de programación utilizado para añadir interactividad a las páginas web.
